ise.utils
=========

Utility functions for data loading, tensor conversion, file I/O, and package
path helpers.

- **functions** — ``get_X_y``, ``get_data``, ``to_tensor``, ``unscale_output``,
  ``unscale_input``, ``get_all_filepaths``, ``load_model``, and helpers for
  uncertainty bands, KDE distributions, and dataset combination.
- **io** — ``check_type`` runtime type validation.
- ``ismip6_model_configs_path`` — absolute path to the bundled ISM configuration
  JSON used by ``ISEFlowAISInputs`` and ``FeatureEngineer``.
